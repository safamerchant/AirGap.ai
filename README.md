# AirGap.ai

AI agent circuit breaker — intercepts, scores, and blocks rogue agent actions.

- `airgap.plain` — Codeplain specification (primary development artefact)
- `index.html` + `support.js` — Codeplain-generated frontend runtime
- `api/evaluate.py` — Vercel Python serverless policy engine

**Live demo:** https://your-airgap.vercel.app
